# Tobi Lütke

CEO of Shopify and author of [[qmd]], a minimal local markdown knowledge base tool
designed as a BM25-based alternative to embedding-heavy RAG systems.

## Relevance to the LLM wiki pattern

Lütke's qmd tool is frequently cited by Andrej Karpathy in the LLM wiki pattern
discussion as an example of how far a simple, curated markdown corpus with keyword
search can take you before you need vector embeddings.

The philosophy behind qmd — "RAG is overengineered for personal knowledge" — aligns
directly with the LLM wiki pattern that [[karpathy-llm-wiki-concept]] describes: use
markdown as the source of truth, keep the index in a simple queryable form, and let
the LLM maintain structure over time.
