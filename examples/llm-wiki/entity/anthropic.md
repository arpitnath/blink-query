# Anthropic

AI safety company founded in 2021, creator of the [[Claude]] model family and the
[[Model Context Protocol]].

## Key facts

- Founded: 2021
- Headquarters: San Francisco, California
- Founders: Dario Amodei, Daniela Amodei, and others formerly from OpenAI
- Notable products: Claude, Claude Code, Model Context Protocol

## Relationship to MCP

Anthropic released the Model Context Protocol as an open specification in November 2024,
published reference server implementations in TypeScript and Python, and integrated MCP
support into Claude Desktop. The protocol was designed to solve the N×M integration
problem between LLM hosts and external tools.
